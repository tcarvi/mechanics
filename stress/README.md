# stress (tensão)

- Tensão de tração ou de compressão
    - Tensão = ( força axial ) / ( área da seção transversal/ortogonal à força axial )

- Tensão normal
     - Tensão = ( força normal ) / ( ... )

- Tensão cisalhamento
     - Tensão = ( força de cisalhamento ) / ( ... )

- Medidas:
    - 1 Pascal = Pa = N / M2 = (newtom) / (metro quadrado)
    - 1 KPa = 1.000 Pa
    - 1 MPa = 1.000.000 Pa
    - 1 GPa = 1.000.000.000 Pa
